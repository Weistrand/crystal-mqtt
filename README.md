# Crystal MQTT

A MQTT communication library for crystal lang


## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     mqtt:
       github: spider-gazelle/crystal-mqtt
   ```

2. Run `shards install`


## Usage

```crystal
require "mqtt"


```
